Criteria for QC pass

  • If size is S then Dimensions should be L: 8,H:6,W:7 (cms) and Weight:250gms
  • If size is M then Dimensions should be L: 9,H:7,W:8 (cms) and Weight:250gms
  • If size is L then Dimensions should be L: 10,H:7,W:12 (cms) and Weight:250gms
  • If size is XL then Dimensions should be L: 12,H:9,W:13 (cms) and Weight:500gms
  • If size is XXL then Dimensions should be L: 12,H:11,W:15 (cms) and Weight:750gms
